# Research

An AI agent workspace for research, content creation, and productivity workflows. Plain-English workflow recipes guide the agent through structured tasks — no code required.

## Structure

| Directory | Purpose |
|-----------|---------|
| `workflows/` | Workflow recipes the agent follows step-by-step |
| `output/` | Finished deliverables (reports, analysis, drafts) |
| `resources/` | Reference docs and templates |

## Workflows

| Workflow | Description |
|----------|-------------|
| `car-shopping.md` | Find and shortlist dealer car listings matching your criteria, with pricing and red flag annotations |
| `research-report.md` | Produce a thorough, well-organized research report on any topic |
